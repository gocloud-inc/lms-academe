## Installation

To install this project just follow the process chronologically.

Clone the project via HTTPS

```bash
git clone https://github.com/gocloud-inc/lms-academe.git
```
## Configuration

Install the node packages

```bash
npm install
```

Start the server 

```bash
npm run dev
```

Build Process

```bash
npm run build
```

Deploy to GH-Pages

```bash
npm run deploy
```
